# Producer API Reference

Complete API reference for Producer (FUZZ-2.0) operations.

For detailed usage examples and guides, see the [Producer Guide](../guides/producer.md).

## Methods

- `create_music()` - All operations (create, extend, cover, replace, swap_vocals, swap_instrumental, variation)
- `upload_music()` - Upload audio files
- `download_music()` - Download in MP3/WAV format
- `get_music()` - Get task status
- `wait_for_completion()` - Wait for task to complete

See the [Producer Guide](../guides/producer.md) for complete documentation and examples.
