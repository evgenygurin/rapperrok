# Suno API Reference

Complete API reference for Suno V4 operations.

For detailed usage examples and guides, see the [Suno Guide](../guides/suno.md).

## Methods

- `create_music()` - Generate music from description
- `create_music_with_lyrics()` - Generate with custom lyrics
- `describe_to_music()` - Generate from description with auto-lyrics
- `extend_music()` - Extend existing tracks
- `concat_music()` - Concatenate multiple clips
- `cover_music()` - Create cover versions
- `stems_basic()` - 2-track stem separation (vocals + instrumental)
- `stems_full()` - 12-track stem separation
- `create_persona()` - Create custom voice model
- `create_persona_music()` - Generate with persona
- `upload_music()` - Upload audio files
- `get_wav()` - Export to WAV format
- `get_midi()` - Export MIDI data
- `get_music()` - Get task status
- `wait_for_completion()` - Wait for task to complete

See the [Suno Guide](../guides/suno.md) for complete documentation and examples.
