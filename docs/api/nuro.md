# Nuro API Reference

Complete API reference for Nuro operations.

For detailed usage examples and guides, see the [Nuro Guide](../guides/nuro.md).

## Methods

- `create_vocal_music()` - Generate music with vocals
- `create_instrumental_music()` - Generate instrumental music
- `get_music()` - Get task status
- `wait_for_completion()` - Wait for task to complete

See the [Nuro Guide](../guides/nuro.md) for complete documentation and examples.
